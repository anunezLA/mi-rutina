# mi-rutina

Automatización semanal de contenido para redes sociales i+Thinnk®

## Estructura

```
mi-rutina/
├── generate_slides.py     # Genera PNG de carruseles Instagram
├── requirements.txt       # Dependencias Python
└── README.md
```

## ¿Qué hace?

1. Lee el Doc "Noticias Semana [fecha]" desde Google Drive
2. Genera 50 slides PNG (5 posts × 5 slides × 2 temas)
3. Sube los PNG a carpeta "Contenido Redes" en Drive

## Temas
- **IA Industrial**: 5 posts con noticias de automatización e IA
- **Animal Feed**: 5 posts con noticias del sector alimento balanceado

## Identidad visual
- Canvas: 1080×1080px
- Colores: Negro #0D0D0D / Dorado #B8995A / Blanco #FFFFFF
- Handle: @ithinnkchile
